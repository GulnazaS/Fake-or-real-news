# Fake-or-real-news
Фальшивые новости — это ложная информация, распространяемая через социальные сети и другие сетевые СМИ для достижения политических или идеологических целей.

В данном проекте была использована библиотека sklearn, с помощью которой удалось построить модель классического машинного обучения с использованием технологий TfidfVectorizer и PassiveAggressiveClassifier.
